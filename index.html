About

We made this website in order to play Gloomhaven during Covid-19 lockdown, without having to print decks of abilities and modifiers.
It's entirely free, without ads, and we hope you enjoy it.
You can check the source code here:
https://github.com/orichalque/yet-an-other-gloomhaven-companion
If you encounter any bug, or have suggestions, please open an issue so that we can correct it asap.
Thanks to:

  • https://github.com/any2cards/gloomhaven

Help

Basics

  • Pick your abilities
  • Select your modifiers
  • Get some gear
  • Customize your abilities with some enhancements
  • Play !

Symbols

  • : Spend item
  • : Recover ability or item
  • : Consume item or lose card
  • : Persistent bonus
  • : Round bonus
  • +1: Use persistent item or ability

Options

  • Set expansion : Access to new classes
  • Accept card : Display a button to add a card from another class to your hand
  • Sort modifiers : Display a button to sort the first X modifiers of your draw pile
  • Quick curse : Display buttons to add a curse or a blessing to your modifiers draw pile
  • Gameplay data saving : Toggle the saving of the state of your board (abilities, modifiers, gear and battle goals)
